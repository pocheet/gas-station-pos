# ⛽ АЗС Терминал — POS система управления наливом топлива

Современный веб-интерфейс для управления топливо-раздаточными колонками (ТРК) на автозаправочных станциях. Приложение взаимодействует с технологическим сервером АЗС через REST API.

---

## 📋 Терминология

| Термин | Обозначение |
|--------|-------------|
| ТРК / Диспенсер | Топливо-раздаточная колонка |
| Пистолет / Nozzle | Раздаточный рукав для отпуска топлива |
| Транзакция | Операция отпуска топлива |
| Пресет | Предустановленная доза (объём или сумма) |

## 🎯 Статусы ТРК

| Код | Статус | Описание | Цвет |
|-----|--------|----------|------|
| 0 | Error | Ошибка / состояние не определено | `#ef4444` |
| 1 | Off | Свободна, ожидание команды | `#4ade80` |
| 2 | Preset | Установлена доза | `#38bdf8` |
| 3 | Call | Пистолет снят | `#818cf8` |
| 4 | CallError | Ошибка: неверный пистолет | `#f87171` |
| 5 | Busy | Идёт налив | `#3b82f6` |
| 6 | BusyOverflow | Налив с переливом | `#f97316` |
| 7 | WaitOffOverflow | Завершение с переливом | `#fbbf24` |
| 8 | WaitOffRemainder | Остаток (ожидание продолжения) | `#f59e0b` |
| 9 | WaitOff | Повесьте пистолет | `#a78bfa` |
| 10 | WaitReset | Ожидание сброса транзакции | `#c084fc` |

## 🔄 API взаимодействие

### Запросы чтения

| Метод | Эндпоинт | Назначение |
|-------|----------|------------|
| GET | `/api/v1/GetConfiguration` | Конфигурация оборудования (продукты, ТРК, пистолеты) |
| GET | `/api/v1/GetFuelEquipmentValues` | Текущие значения (статусы, транзакции, счётчики) |

### Управление ТРК

Все операции выполняются в последовательности **Lock → Действие → Unlock**:

| Действие | Эндпоинт | Когда применяется |
|----------|----------|-------------------|
| Запуск налива | `StartPumpTransaction` | Статус OFF |
| Остановка | `StopPumpTransaction` | Статус BUSY |
| Продолжение | `ContinuePumpTransaction` | Статус WaitOffRemainder |
| Сброс (завершение) | `ResetPumpTransaction` | Статусы WaitOffRemainder, WaitReset |

## 🎨 Пользовательский интерфейс

### Тёмная тема

- Основной фон: `#0a0a14`
- Фон панелей: `#1a1a2e`
- Фон карточек: `#0f3460`
- Акцентный цвет: `#00d4aa` (бирюзовый)

### Макет
```
┌─────────────────────────────────────────────────────────┐
│ TopNavbar (дата и время по центру) │
├──────────┬──────────────────────────┬───────────────────┤
│ │ │ │
│ Pump │ TransactionInfo │ PaymentMethod │
│ Sidebar │ (статус, пистолет, │ Selector │
│ (ТРК / │ показатели, │ │
│ писто- │ кнопки управления) │ PresetKeyboard │
│ леты) │ │ (цифровая │
│ │ OrderTable │ клавиатура) │
│ │ (заказы) │ │
│ │ │ │
├──────────┴──────────────────────────┴───────────────────┤
│ Footer (кнопка "Настройка цены") │
└─────────────────────────────────────────────────────────┘
```

### PumpSidebar

- Режим **"Диспенсеры"**: карточки всех ТРК в сетке 2×N
- Режим **"Диспенсер"**: пистолеты выбранной ТРК
- При клике на ТРК → авто-переход в режим "Диспенсер"
- Цветовая индикация статусов

### TransactionInfo

Единый контейнер, отображающий:
- Номер ТРК и статус
- Выбранный пистолет (номер и продукт)
- Объём и сумма (из PresetKeyboard или текущей транзакции)
- Прогресс-бар при наливе
- Три кнопки: `Запуск` / `Остановить` / `Завершить`

### OrderTable

- Таблица-корзина: всегда 3 строки
- Автоматически заполняется при вводе параметров
- Клик по строке восстанавливает параметры
- Крестик удаляет заказ и сбрасывает форму

### PresetKeyboard

- Цифровая клавиатура 4×5
- Кнопка `Ввод` запускает налив и фиксирует заказ
- Кнопка `C` сбрасывает введённое значение
- Динамический цвет дисплея: зелёный (объём) / золотой (сумма)
