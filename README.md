# ⛽ АЗС Терминал - POS система управления наливом топлива

Современный веб-интерфейс для управления топливо-раздаточными колонками (ТРК) на автозаправочных станциях.

## 🚀 Возможности

- 📊 Мониторинг состояния ТРК в реальном времени
- ⛽ Управление наливом топлива
- 📈 Визуализация процесса налива
- 🎨 Современный dark theme интерфейс
- 📱 Адаптивный дизайн
- 🔄 Автоматическое обновление данных (polling 1 сек)

## 🛠 Технологии

- **React 18** + TypeScript
- **Vite** - сборщик
- **TanStack Query** - управление серверным состоянием
- **Axios** - HTTP клиент
- **CSS3** - стилизация с анимациями

## 📋 Статусы ТРК

| Код | Статус | Описание |
|-----|--------|----------|
| 0 | Error | Ошибка/состояние не определено |
| 1 | Off | Ожидание команды |
| 2 | Preset | Установлена транзакция |
| 3 | Call | Пистолет снят |
| 4 | CallError | Ошибка: неверный пистолет |
| 5 | Busy | Идет налив |
| 6 | BusyOverflow | Налив с переливом |
| 7 | WaitOffOverflow | Ожидание завершения с переливом |
| 8 | WaitOffRemainder | Ожидание продолжения |
| 9 | WaitOff | Ожидание возврата пистолета |
| 10 | WaitReset | Ожидание сброса транзакции |

cd gas-station-pos
